## Dev environment

Run `git submodule init` && `git submodule update` after checking out the repository

redash 

* `npm install`
* `make up`
* `docker-compose run --rm server create_db`
* `npm run start`
* `open http://localhost:5000`

turnilo

* `cd turnilo`
* `npm install`
* `npm run build:dev`
* `npm run start:dev:examples`
* `open http://localhost:9090`
